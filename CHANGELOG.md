# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure for frontend and backend.
- Core knowledge management features: Capture, Process, and Note Detail.
- Task vs Collection separation logic.
- Clickable header titles and task navigation.
- User avatar dropdown menu in the Home screen.
- Standard project documentation (README, CONTRIBUTING, LICENSE, etc.).

### Changed
- Refined task completion logic to avoid accidental clicks.
- Improved markdown viewer layout.

### Fixed
- Fixed bug where empty full notes would show only the section label.
- Resolved and prevented double back arrows in task/collection headers.
- Fixed modal transparency not covering the status bar on Android.
