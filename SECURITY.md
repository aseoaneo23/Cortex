# Security Policy

If you believe you've found a security vulnerability in Cortex, please report it to us as soon as possible.

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.** Instead, email us at security@cortex-app.io (dummy).

## Disclosure Policy

1.  We will acknowledge receipt of your report within 48 hours.
2.  We will investigate the issue and let you know if we need more information.
3.  We will provide a fix (or a mitigation) as quickly as possible.
4.  Once the fix is ready, we will coordinate a public disclosure with you.

Thank you for helping keep Cortex secure!
